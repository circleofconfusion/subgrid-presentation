display: contents vs. subgrid

  • display: contents removes element's box from the layout
  • element with display: contents can still pass on inheritable rules like color
  • Cannot pass on rules like background-color
  • subgrid keeps semantic role of element
  • allows element's children to become grid items of parent grid
  • Retains box and box styling in layout
